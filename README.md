# Tristan Martin — Personal Landing Page

A minimalist, interactive personal landing page serving as the central hub for my online presence and creative work.

## Purpose

A unified gateway that directs visitors to my creative and professional platforms through a clean, modern layout. Built with Next.js and Framer Motion for a fluid, responsive user experience.

## Features

- **Fluid Animations** — Smooth entry and interactive spring animations powered by Framer Motion.
- **Responsive Stack Layout** — Headshot, bio, and cleanly organized link tiles that adapt seamlessly between mobile and desktop devices.
- **Minimalist Aesthetic** — Clean typography and subtle hover states for a focused user experience.

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org) (App Router)
- **Language:** TypeScript
- **Styling:** [Tailwind CSS](https://tailwindcss.com) v4
- **Animation:** [Framer Motion](https://www.framer.com/motion/)
- **Fonts:** [Satoshi](https://www.fontshare.com/fonts/satoshi) (headings & body) + [Sentient](https://www.fontshare.com/fonts/sentient) (serif accents) via Fontshare
- **Deployment:** Vercel

## Getting Started

```bash
cd tristan-landing
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.
